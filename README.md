## prog1-project

Der Sourcecode und diese Beschreibung ist auch auf GitHub verfügbar (schöner zum Lesen) :smile:

http://raeffu.github.io/prog1-project/

### Ziel

Erstellen einer Android Applikation, die Sudokus lösen kann.

### Anleitung

1. SudokuSolver.apk auf einem Android Smartphone installieren und öffnen.
   * z.B. per Download von Github
   (http://github.com/raeffu/prog1-project/blob/master/SudokuSolver.apk?raw=true)
   * oder über Dropbox
   (...Prog1&2/Projects/Projekte/SudokuSolver_Laubscher)
   * oder per Dateiübertragung.
2. Nach Download auf Smartphone das *.apk File öffnen und installieren.
3. Sudoku App starten.
4. Ein Beispiel-Sudoku wird angezeigt und kann mit Klick auf "Solve" gelöst werden.

Es können auch andere Sudokus eingegeben werden (als Zeichenkette, siehe Beispielsudoku).
